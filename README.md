# DBTS

Hi Everyone, I'm sanjeev arora developer of this application. This application is developed to track the school and college buses and some extra features are also included :
1. In-App Messaging 
2. Live Tracking 
3. Delay Status
4. Fee Payment 
5. Nearest Stop Suggestor
